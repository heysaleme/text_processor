

# 📘 go-reloaded

**go-reloaded** — это утилита на Go, которая "пересобирает" текст:
исправляет пробелы и пунктуацию, корректирует артикли **a/an** и применяет встроенные команды форматирования — вроде `(up)`, `(low)`, `(cap)`, `(bin)`, `(hex)` — прямо внутри текста.

---

## ✨ Возможности

🔹 **Нормализация пробелов и пунктуации**
    ◦ Удаляет лишние пробелы перед знаками препинания
    ◦ Добавляет пробел после запятых и точек при необходимости

🔹 **Автоматическое исправление артиклей**
    ◦ `a apple` → `an apple`
    ◦ `an hat` → `a hat`

🔹 **Поддержка встроенных команд форматирования**
Скобочные команды применяются к последним словам:

| Команда        | Действие                                 |
| -------------- | ---------------------------------------- |
| `(up)`         | Верхний регистр                          |
| `(low)`        | Нижний регистр                           |
| `(cap)`        | Первая буква заглавная                   |
| `(hex)`        | Перевод из HEX в десятичную систему      |
| `(bin)`        | Перевод из BIN в десятичную систему      |
| `(команда, n)` | Применяет команду к `n` последним словам |

🔹 **Запуск из терминала**
    ◦ Без сторонних библиотек, только стандартный Go.

## 🚀 Установка и запуск

#### 1️⃣ Клонируй репозиторий

```bash
git clone https://github.com/aaristan/go-reloaded.git
cd go-reloaded
```

#### 2️⃣ Подготовь входной файл

Создай текстовый файл, например `sample.txt`:

```
harold wilson (cap, 2) : ' I am a optimist ,but a optimist who carries a raincoat . '
```

#### 3️⃣ Запусти программу

```bash
go run . sample.txt result.txt
```

* ✅ Первый аргумент — путь к **входному** файлу
* ✅ Второй аргумент — путь к **выходному** файлу

После выполнения будет создан (или перезаписан) `result.txt`.

#### 4️⃣ Посмотри результат

```bash
cat result.txt
```

**Вывод:**

```
Harold Wilson: 'I am an optimist, but an optimist who carries a raincoat.'
```

## 📋 Примеры

| Входной текст                                | Результат                        |
| -------------------------------------------- | -------------------------------- |
| `it (cap) was a amazing (cap) experience ?!` | `It was an Amazing experience?!` |
| `A a apple`                                  | `A an apple`                     |
| `an hat`                                     | `a hat`                          |
| `Hello (up)`                                 | `HELLO`                          |
| `1101 (bin)`                                 | `13`                             |
| `1f (hex)`                                   | `31`                             |

---

## ⚙️ Логика работы

1. **Токенизация** — разбивка текста на слова, знаки и команды
2. **Применение команд** — `(up)`, `(cap)`, `(bin)` и т.д.
3. **Коррекция артиклей** — с учётом контекста (гласные/согласные)
4. **Реконструкция текста** — восстановление с учётом правил пунктуации и пробелов

---

## ❗ Примечания

* Программа принимает **ровно два аргумента** — путь к входному и выходному файлу
* В случае неверного вызова программа выведет подсказку:

```
Usage: go run . <input.txt> <output.txt>
```

* Все изменения (артикли, команды, пробелы, пунктуация) применяются последовательно и записываются в выходной файл
